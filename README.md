# floatText
jQuery Plugin for floating text in blocks

 * Example:
```html
<!-- HTML -->
<div class="classForInit familyName">Put text here</div>
<div class="familyName"></div>
<div class="familyName"></div>
<div class="familyName"></div>
<div class="familyName"></div>
```


```js
//Javascript
$(function(){
    $('.classForInit').floatText({
        familyClassName: '.familyName'
    });
});
```
